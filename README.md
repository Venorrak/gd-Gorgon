# gd-Gorgon
Gorgon is a high-performance Godot plugin for building dynamic UI graphs, skill trees, and node-based webs.

## This is a more bare bones version of the original [Medusa](https://github.com/arukurei/Medusa) plugin, some bugs (or things I just didn't like how they worked) were fixed and I added some features like the ability to use custom highlighting patterns.
